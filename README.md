# git을 배워보자
## git 설치
### 1. [git-scm.com](https://git-scm.com) 이동하여 우측 중앙의 다운로드 버튼을 클릭하고 설치한다(기본값으로 설치)
### 2. [github.com](https://github.com) 으로 이동한다.
### 3. 우측 상단의 **signup** 버튼을 클릭한다.
### 4. 회원 가입을 완료하고 등록한 이메일로 이동하여 velified 버튼을 클릭하여 인증한다.
